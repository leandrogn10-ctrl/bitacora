    Auto-push code to GitHub Pages

    Push an updated bitacora.html to your repo without leaving the app. After pushing, your phone gets the new version on next refresh (~30 sec). Requires a separate fine-grained token with Contents: Read & write on the bitacora repo only.

    Setup once: at github.com/settings/personal-access-tokens/new → Repository access: Only select repositories → pick bitacora → Permissions → Repository permissions → Contents: Read and write → Generate token. Copy the github_pat_… string.

    Cloud sync (GitHub Gist)

    Your data lives in a private gist on your GitHub account. Last-write-wins between devices.

    Step 1. Open GitHub token page (pre-configured). Click Generate token, copy the ghp_… string.
    Step 2. Paste below. Leave Gist ID blank on first device — click "Enable & sync" to create.
    Step 3. On other devices, paste the same token + the gist ID shown after step 2.
