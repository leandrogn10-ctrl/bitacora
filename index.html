    Cloud sync (GitHub Gist)

    Your data lives in a private gist on your GitHub account. Last-write-wins between devices. Setup is one click + paste.

    Step 1. Open GitHub token page (pre-configured). Click Generate token at the bottom, then copy the ghp_… string.
    Step 2. Paste below. Leave Gist ID blank on your first device — click "Enable & sync" to create.
    Step 3. On other devices, paste the same token + the gist ID shown after step 2.
